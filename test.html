Catatan
Ada 5 hasil untuk taruhan Mix Parlay: 1. W = Win ( Odds dihitung semuanya )
2. WH = Win Half ( Odds dibagi )
3. D = Draw ( Odds tidak hitung )
4. LH = Lose Half ( Odds menjadi 0.5 )
5. L = Lose ( Odds Menjadi 0 )

 Rumus menghitung Betting parlay: ( Total semua perkalian odds ) - 1 kemudian dikalikan dengan Nilai Bet
Cara menghitung jika semua taruhan W ( WIN / menang )
Jika taruhan WIN atau menang semua, maka semua odds di kalikan lalu di kurang 1, kemudian di kalikan dengan Nilai Bet anda.
Cara menghitung jika ada taruhan yang WH ( Win Half / Menang Setengah )
Jika ada taruhan Win Half atau Menang Setengah, maka odds pertandingan tersebut menjadi di kurang 1 di bagi 2 kemudian di tambah 1.
Cara menghitung jika ada taruhan yang D ( DRAW / Seri )
Jika ada taruhan DRAW atau seri, maka odds pertandingan tersebut tidak dihitung.
Cara menghitung jika ada taruhan yang LH ( Lose Half / Kalah Setengah )
Jika ada taruhan Lose Half atau Kalah Setengah, maka odds pertandingan tersebut menjadi 0.5.
Cara menghitung jika ada taruhan yang L ( Lose / Kalah )
Jika ada taruhan Lose atau kalah, maka semua taruhan anda akan di hitung kalah.
